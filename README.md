# Counter redux
It's a simple project using Redux pattern since 0
 
### Tech

This project was developed with:

* [Typescript] - 
* [Node] -
* [ts-node] -


### Requirements
[Node](https://nodejs.org/) v14.17.3

### Instalation

```sh
$ cd folder
$ npm install or npm i
```

### Run
Pasar de TS a JS en tiempo real (en una terminal)
```sh
$ ts-node app
```
